## KEY CVARS 2026

### FPS & Performance
```
r.Kuro.MaxFPS.ThirdParty120=1     # Unlock 120 FPS
r.VSync=0                         # Disable VSync
r.ScreenPercentage=85             # Balance
r.HZBOcclusion=1                 # Fix cave white spots
```

### Quality Settings
```
r.ShadowQuality=3                # Medium-High shadows
r.Mobile.KuroPostprocess=1       # Character rim light
r.Kuro.NpcDisappearDistance=20000 # Fix NPC pop-in
r.ViewDistanceScale=3             # Global view distance
r.MaxAnisotropy=16               # Best texture quality
```

### Mobile Specific
```
r.Mobile.DeviceEvaluation=3       # Force flagship tier
r.MobileContentScaleFactor=0.0    # Native resolution
r.Kuro.AutoCoolEnable=1          # Thermal protection
r.Mobile.WaterSSR=1              # Water reflections
```

### 2026 New CVars
```
# NVIDIA Reflex (lower input lag)
t.Streamline.Reflex.Enable=1
t.Streamline.Reflex.Mode=1

# DLSS / Upscaling
r.NGX.Enable=1
r.NGX.DLSS.Quality=3

# Vulkan fixes
r.Vulkan.RobustBufferAccess=1
r.Vulkan.StencilBuffer=1

# Streaming
r.Streaming.UsingNewKuroStreaming=1
r.Streaming.PoolSize=600
r.Streaming.MinBoost=4.0
```

## BUG FIXES REFERENCE

| Issue | Fix CVar |
|-------|----------|
| White spots in caves | r.HZBOcclusion=1 |
| NPC pop-in at distance | r.Kuro.NpcDisappearDistance=20000 |
| Animation stuttering | a.URO.ForceAnimRate=1 |
| Thermal throttling | r.Kuro.AutoCoolEnable=1 |
| Low resolution (Android) | r.MobileContentScaleFactor=0.0 |
| Can't reach 120 FPS | r.Kuro.MaxFPS.ThirdParty120=1 |
| No Ultra option | r.Kuro.GraphicsQuality.ThirdPartyUltraEnable=1 |
| Network lag | LGUIBlurTexture.save=0 |
| Blurry textures | r.MipMapLODBias=-1 + r.MaxAnisotropy=16 |
| Vulkan crashes | r.Vulkan.RobustBufferAccess=1 |
| Input lag (PC) | r.VSync=0 + t.Streamline.Reflex.Enable=1 |
