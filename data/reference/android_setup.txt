ANDROID SETUP & DEPLOYMENT GUIDE
Tool: P42 WuWa Config File Setup Toolkit
==================================================================

REQUIRED APPS
-------------
1. Termux - Terminal emulator for Android
   Download: F-Droid or Google Play
2. Shizuku - Permission manager
   Download: Google Play
3. rish + rish_shizuku.dex files (from P42 DC/TG channel)

SETUP STEPS (First Time)
-------------------------
Step 1: Install Termux and Shizuku from Play Store
Step 2: Create folder named "P42" inside Downloads
Step 3: Open Termux, run: termux-setup-storage (grant permission when asked)
Step 4: Place Engine.ini, DeviceProfiles.ini, rish, rish_shizuku.dex in P42 folder
Step 5: Android 14+ Fix - run in Termux:
  cd ~/storage/downloads/P42 && cp rish rish_shizuku.dex /data/data/com.termux/files/usr/bin/ && chmod +x /data/data/com.termux/files/usr/bin/rish /data/data/com.termux/files/usr/bin/rish_shizuku.dex

DEPLOYMENT COMMANDS
--------------------
Navigate to P42 folder (run every new Termux session):
  cd ~/storage/downloads/P42

Backup current configs:
  rish -c "cp '/storage/emulated/0/Android/data/com.kurogame.wutheringwaves.global/files/UE4Game/Client/Client/Saved/Config/Android/DeviceProfiles.ini' '/storage/emulated/0/Download/P42/DeviceProfiles.ini.bak'" && rish -c "cp '/storage/emulated/0/Android/data/com.kurogame.wutheringwaves.global/files/UE4Game/Client/Client/Saved/Config/Android/Engine.ini' '/storage/emulated/0/Download/P42/Engine.ini.bak'"

Delete old configs:
  rish -c "cd '/storage/emulated/0/Android/data/com.kurogame.wutheringwaves.global/files/UE4Game/Client/Client/Saved/Config/Android' && rm -f DeviceProfiles.ini Engine.ini"

Deploy new configs:
  rish -c "cp /storage/emulated/0/Download/P42/DeviceProfiles.ini /storage/emulated/0/Android/data/com.kurogame.wutheringwaves.global/files/UE4Game/Client/Client/Saved/Config/Android/" && rish -c "cp /storage/emulated/0/Download/P42/Engine.ini /storage/emulated/0/Android/data/com.kurogame.wutheringwaves.global/files/UE4Game/Client/Client/Saved/Config/Android/" && rm DeviceProfiles.ini Engine.ini

Copy game log:
  rish -c "cp '/storage/emulated/0/Android/data/com.kurogame.wutheringwaves.global/files/UE4Game/Client/Client/Saved/Logs/Client.log' '/storage/emulated/0/Download/P42/Client.log'"

Restore backup:
  for f in DeviceProfiles.ini Engine.ini; do cat "$f.bak" | rish -c "cat > '/storage/emulated/0/Android/data/com.kurogame.wutheringwaves.global/files/UE4Game/Client/Client/Saved/Config/Android/$f'"; done

ANDROID CONFIG FILE PATHS
--------------------------
Config files: /storage/emulated/0/Android/data/com.kurogame.wutheringwaves.global/files/UE4Game/Client/Client/Saved/Config/Android/
Log file: /storage/emulated/0/Android/data/com.kurogame.wutheringwaves.global/files/UE4Game/Client/Client/Saved/Logs/Client.log

ANDROID 14 SHIZUKU ISSUES
--------------------------
Error: "writable dex" 
Fix: Copy rish files to Termux bin directory (see setup step 5)

Shizuku not running:
1. Open Shizuku app
2. Tap "Start via Wireless debugging" 
3. Enable wireless debugging in Developer Options first

ANDROID PERFORMANCE NOTES
--------------------------
- 120 FPS requires r.Kuro.MaxFPS.ThirdParty120=1 in DeviceProfiles.ini
- Native resolution: r.MobileContentScaleFactor=0.0
- Force highest device tier: r.Mobile.DeviceEvaluation=3
- Thermal protection: r.Kuro.AutoCoolEnable=1 (recommended)
- Vulkan stability mode: r.Vulkan.RobustBufferAccess=1 (if crashes)

READING CLIENT.LOG FOR DEVICE INFO
------------------------------------
The Client.log file contains:
- Device GPU info (search for "GPU" or "Adapter")
- Device memory info
- Active device profile name
- Rendering API (Vulkan/OpenGL)
- Detected device tier
- Current CVars loaded

SUPPORTED DEVICES (from AlteriaX configs)
------------------------------------------
Snapdragon 8 Gen 1/2/3: Android_Adreno7xx profile
Snapdragon 888/870: Android_Adreno6xx_High
Snapdragon 865/855: Android_Adreno6xx
Dimensity 9000+: Android_Default (maps to VeryHigh)
Exynos flagships: Check Android_Default profile
