P42 WuWa AI Knowledge Base - Data Folder
==================================================================

HOW TO ADD MORE DATA
---------------------
Drop any .txt files into this folder to expand the AI's knowledge.
The AI bot will automatically read all .txt files in this folder
when answering questions.

INCLUDED FILES
--------------
- ue4_commands.txt      UE4 console commands & CVars reference
- wuwa_configs.txt      WuWa-specific Kuro commands & fixes
- android_setup.txt     Android deployment & Shizuku guide
- README.txt            This file

TIPS FOR ADDING DATA
---------------------
1. Name files descriptively (e.g., "my_custom_settings.txt")
2. Use plain text format
3. Include headers and labels for better search accuracy
4. Add command=value pairs on separate lines for best results
5. You can paste entire config files as .txt

EXAMPLE CUSTOM FILE
--------------------
Create a file like "my_phone_settings.txt":

  Device: Samsung Galaxy S24
  GPU: Adreno 750
  Best settings:
  r.Mobile.DeviceEvaluation=3
  r.Kuro.MaxFPS.ThirdParty120=1
  FPS: 120 works great on this device

The AI will use this info when answering questions about your device.
